DivisAR

DivisAR es una aplicación web que muestra las cotizaciones de diferentes divisas en pesos argentinos en tiempo real. Utiliza la API de Dolar API para obtener valores actualizados de monedas como el dólar, euro, real, peso chileno y peso uruguayo. La interfaz es una SPA (Single Page Application), diseñada para ofrecer una experiencia rápida y dinámica, permitiendo a los usuarios seleccionar distintas monedas y ver su cotización actual, tanto de compra como de venta, junto con la fecha de la última actualización.

Sistema Clínico

Este sistema web está diseñado para la gestión eficiente de pacientes en una clínica. Permite realizar operaciones CRUD para agregar y administrar pacientes, asegurando un flujo de trabajo organizado. Incluye un sistema de inicio de sesión y registro para administradores, donde el administrador general aprueba los nuevos registros de usuarios. Además, facilita la creación de citas en fechas y horarios disponibles, optimizando la programación y mejorando la atención al paciente. Una solución integral para la gestión clínica moderna.

API del Sistema Clínico

Esta API, desarrollada en Go, permite la interacción con el sistema web de gestión de pacientes. Utiliza las librerías GORM para la gestión de bases de datos MySQL, Gorilla Sessions para manejar sesiones de usuario de manera segura, y bcrypt para la encriptación de contraseñas. Además, se basa en el framework Gin-Gonic para proporcionar un rendimiento óptimo en la gestión de rutas y respuestas. La API ofrece endpoints para operaciones CRUD en pacientes y la gestión de citas, garantizando una integración fluida y segura con el sistema web.
